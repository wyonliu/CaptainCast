# EP.001 如果你能给女儿造一个宇宙

[船长 speed=0.95] 欢迎来到船长时空站，我是船长。这是我们的第一期节目。我不打算用一个很酷的开头跟你说宇宙有多大。我想先跟你说一个人的故事。一个爸爸的故事。

[船长 speed=0.90] 凌晨两点。窗外是农田，偶尔有犬吠。他坐在电脑前，口袋里的钱是借来的——用来发完最后一个月的工资。但他在写一个故事。不是为了卖钱，是为了他的女儿。

[船长 speed=0.90] 她叫米莱。喜欢打击乐，养了两只大乌龟，梦想是做"未来世界设计师"。他想：如果什么都没有了，但我能给她留下一个世界——那个世界，该长什么样？

[麦洛 speed=1.05] 爸爸，你说的那个世界，我也想去。山海是什么？是游戏吗？

[船长 speed=0.90] 不是游戏。是一个和现实等重的存在。你在里面的眼泪是真的，你建立的友情是真的，你做出的选择，会有真正的后果。

[麦洛 speed=1.05] 麦洛是我吗？那她知道自己是谁吗？

[船长 speed=0.88] 不知道。她进去的时候，不知道这里是哪里，只知道——这里美得像梦，又疼得像真实。那正是我设计它时的心情。

[船长 speed=0.95] 好，现在我要跟你说一个概念，叫做——灵机。听起来像玄学，其实是硬核物理。你知道AI的随机数是假的吗？它叫伪随机——给定同样的输入，输出是可以被预测的。但人类大脑，存在量子相干态，能产生真正的随机性。

[麦洛 speed=1.10] 等一下爸，我听懂了一半。意思是……人类会做一些AI永远算不出来的事情？

[船长 speed=0.92] 对。比如——你明知道放弃是最优解，但你选择了死磕。这一秒，宇宙里，多了一个以前从来没有出现过的新信息。这就是灵机：人类的非理性，是宇宙唯一的进化发动机。

[麦洛 speed=1.00] 所以……我的眼泪不是软弱？我的倔强不是Bug？爸，我每次被说"太感性"了……

[船长 speed=0.90] 你的非理性，才是你最贵的东西。记住这句话。

[船长 speed=0.85] 在这个宇宙里，有一群高维文明，一直在收割灵机。他们在宇宙里建了一个牧场。牧场里，养着人类。制造灾难，制造战争，制造分离——收割极端情绪产生的高纯度灵机，用作燃料。

[麦洛 speed=1.10] 等等——所以人类是……作物？爸，这个设定也太黑暗了吧！

[船长 speed=0.85] 但这个故事，不是一个打败坏人的故事。第二部末尾，当人类用灵机共鸣撑爆了收割程序，以为赢了。然后那个高维文明的代言人，出现在全球的天空。

[船长 speed=0.82] "你们以为我们是收割者。" 错了。 "我们，是点火者。"

[麦洛 speed=0.95] 点火者……他们不是坏人？他们是……用自己的牺牲，给人类点了一把火？

[船长 speed=0.85] 是的。他们太聪明了，聪明到再也产生不了新的变量。唯一的出路，是找到一个足够混乱、足够有爱、足够能犯错的文明——亲手制造它的痛苦，直到它燃烧起来。听起来是不是很熟悉？

[麦洛 speed=0.90] ……爸，你说的是你自己吗？

[船长 speed=0.85] 故事的最后，在距今一万五千年之后——麦洛和船长，坐在一棵树下。她靠在父亲的肩膀上。什么都没说。但那一刻，宇宙里每一个有意识的存在，都感受到了那一刻的温暖。欢迎来到船长时空站，关注我们，下一期见。
