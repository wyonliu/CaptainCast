## 一、选题与核心立意

### 候选主题（推荐首选）

**主题 A：「灵机」从何而来——量子意识的边界在哪里** ⭐ 推荐
- 核心：从 EP.001 的灵机理论深挖，讲"意识"这件事
- 科学支撑：彭罗斯-哈莫罗夫量子脑理论（Orch-OR）、自由意志的神经科学
- 情感锚：船长在至暗时刻"明知继续是徒劳，还是继续"——这是灵机的最高表达
- 麦洛视角：她在山海里第一次意识到"我是谁"——这个时刻的震撼

**主题 B：宇宙牧场设计师——如果你是减一文明，你会怎么做**
- 核心：换位思考，带听众设计"痛苦"——什么样的痛苦才能让文明进化
- 科学支撑：压力-成长曲线、人类文明史上的至暗与突破（黑死病→文艺复兴）
- 情感锚：船长的2019-2024，是他的"被收割期"，结果产生了山海
- 互动设计：让听众说出自己的"被收割时刻"，发现它变成了什么

**主题 C：2032年的第一天——麦洛进入山海**（世界观深挖）
- 核心：详细描述麦洛第一次进入山海的感受——视听联觉体验
- 适合做成更沉浸式的"声景播客"，配上专门的背景音乐
- 情感锚：女儿真实的"未来世界"想象

### 推荐：主题 A + 引子 C
- 用麦洛进入山海的场景切入（30秒片段）
- 然后展开"意识/灵机"的科学讨论
- 结尾回到父女树下，和EP.001呼应

## 四、BGM 选择（听众建议已加入）

### 要求
- 免版权（CC0 / CC BY）
- 不抢戏，烘托气氛
- 符合"敦煌赛博朋克"调性：东方感 + 科技感 + 神秘感
- 不能是旋律感太强的商业音乐

### 推荐来源
- **Pixabay Music**：搜索 "sci-fi ambient" / "meditation ambient" / "space"
- **Free Music Archive (FMA)**：搜索 "space ambient" / "Chinese traditional"
- **ccMixter**：搜索 "cinematic ambient"

### 具体推荐（待船长选定）
- 主基调：低调的合成器垫底（synth pad），若隐若现的中国笛声
- 片头片尾：稍有起伏，带宇宙感
- 中间对话：静音或只留极轻微背景音（避免与对话竞争）

### 加 BGM 的技术方案（gen_video.py 后续）
- 在 gen_voice.py 合并时，用 pydub 叠加 BGM 轨（-18dB 以下）
- 片头3秒纯BGM，然后淡入对话
